## Job Application Tracker

A full-stack application that helps users keep track of job applications, application statuses, interview stages, and follow-up notes.

### Features
- Create and manage job applications
- Track application status (Applied, Interview, Rejected, Offer)
- Store notes and follow-up dates
- Simple dashboard overview

### Tech Stack
- Frontend: React / Next.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

### Why this project?
Job hunting can get messy fast. This app helps users stay organized and focused by keeping everything in one place.
